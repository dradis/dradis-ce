# Echo: AI copilot for Dradis

The add-on requires [Dradis CE](https://dradis.com/ce/) > 4.0, or [Dradis Pro](https://dradis.com/).


## Install

Add this to your `Gemfile.plugins`:

    gem 'dradis-echo'

And

    bundle install

Restart your Dradis server and you should be good to go.
